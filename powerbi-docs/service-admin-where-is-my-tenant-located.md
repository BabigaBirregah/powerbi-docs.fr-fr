# <a name="where-is-my-power-bi-tenant-located"></a>Où est situé mon client Power BI ?

<iframe width="560" height="315" src="https://www.youtube.com/embed/0fOxaHJPvdM?showinfo=0" frameborder="0" allowfullscreen></iframe>

Découvrez où se trouve votre client Power BI et comment l’emplacement est sélectionné. Cette notion est importante, car elle peut affecter les interactions de formation que vous avez avec le service.

## <a name="how-to-determine-where-your-power-bi-tenant-is-located"></a>Déterminer où est situé votre client Power BI

Pour rechercher la région où se trouve votre client, procédez comme suit.

1. Dans le service Power BI, menu supérieur, sélectionnez l’aide ( **?** ), puis **À propos de Power BI**.

1. Recherchez la valeur suivante à côté de **Vos données sont stockées dans**. Il correspond à la région où se trouve votre client. La valeur est également la région où sont stockées vos données, sauf si vous utilisez des capacités dédiées dans des régions différentes pour vos espaces de travail.

    ![Région de données](media/service-admin-where-is-my-tenant-located/power-bi-data-region.png)

## <a name="how-the-data-region-is-selected"></a>Méthode de sélection de la région de données

La région de données est basée sur le pays que vous sélectionnez lors de la création du client. La sélection s’applique pour vous inscrire pour les deux Office 365 et à Power BI, étant donné que ces informations sont partagées. S’il s’agit d’un nouveau client, sélectionnez le pays approprié dans la liste lorsque vous vous inscrivez.

![Sélection du pays](media/service-admin-where-is-my-tenant-located/sign-up-country-selection.png)

Power BI choisit une région de données le plus proche de votre sélection, ce qui détermine où les données sont stockées pour votre client.

> [!IMPORTANT]
> Vous ne pouvez pas modifier la sélection après avoir créé le locataire.
