# <a name="sign-up-for-power-bi-as-an-individual"></a>S’inscrire à Power BI en tant qu’individu

Power BI peut servir aussi bien d’outil de rapport et de visualisation personnel que de moteur d’analytique et de décision pour des projets de groupe, des divisions ou des grandes entreprises. Cet article explique comment obtenir un essai Power BI Pro en tant qu’individu. Si vous êtes un administrateur Power BI, consultez [gestion des licences Power BI dans votre organisation](service-admin-licensing-organization.md).

![tableau de bord Power BI](media/service-self-service-signup-for-power-bi/dashboard.png)

## <a name="supported-email-addresses"></a>Adresses e-mail prises en charge

Avant de commencer le processus d’inscription, il est important de savoir quels types d’adresses de messagerie que vous pouvez utiliser dans Power BI :

* L’inscription à Power BI nécessite l’utilisation d’une adresse e-mail professionnelle ou scolaire. Vous ne pouvez pas vous inscrire à l’aide des adresses e-mail fournies par les services de messagerie de consommateur ou fournisseurs de télécommunication. Cela inclut outlook.com, hotmail.com, gmail.com et autres.

* Une fois que vous êtes connecté, vous pouvez [inviter des utilisateurs invités](https://docs.microsoft.com/azure/active-directory/active-directory-b2b-what-is-azure-ad-b2b) à voir votre contenu Power BI avec n’importe quelle adresse e-mail, notamment des comptes personnels.

* Vous pouvez vous inscrire à Power BI avec des adresses .gov ou .mil, mais celles-ci nécessitent un processus différent. Pour plus d’informations, consultez [inscrire votre organisation US Government dans le service Power BI](service-govus-signup.md).

## <a name="sign-up-for-a-trial"></a>S’inscrire à un essai

Effectuez les étapes suivantes pour vous inscrire à un essai Power BI Pro. Consultez la section suivante, [expiration de la version d’évaluation](#trial-expiration), pour en savoir plus sur les options, vous avez lors de cette version d’évaluation expire.

1. Accédez à la [page d’inscription](https://signup.microsoft.com/signup?sku=a403ebcc-fae0-4ca2-8c8c-7a907fd6c235) dans Power BI.

1. Entrez votre adresse e-mail, puis sélectionnez **S’inscrire**.

    ![Commencer](media/service-self-service-signup-for-power-bi/get-started.png)

1. Si vous obtenez un message similaire à celui-ci, choisissez une option pour recevoir un code de vérification, puis passez à l’étape suivante de cette procédure.

    ![Prouver que vous n’êtes pas un robot](media/service-self-service-signup-for-power-bi/prove-robot.png)

    Si vous obtenez un message similaire à celui-ci, terminez les étapes pour se connecter et utiliser Power BI.

    ![Compte existant](media/service-self-service-signup-for-power-bi/existing-account.png)

1. Entrez le code que vous avez reçu, puis sélectionnez **S’inscrire**.

    ![S’inscrire](media/service-self-service-signup-for-power-bi/sign-up.png)

1. Vérifiez votre courrier électronique un message similaire à celle-ci.

    ![Vérification par e-mail](media/service-self-service-signup-for-power-bi/email-verification.png)

1. Sur l’écran suivant, entrez vos informations et le code de vérification de l’e-mail. Sélectionnez une région, passez en revue les termes et conditions dont les liens figurent sur cet écran, puis sélectionnez **Démarrer**.

    ![Créer un compte](media/service-self-service-signup-for-power-bi/create-account.png)

1. Vous êtes alors redirigé vers [page de connexion Power BI](https://powerbi.microsoft.com/landing/signin/), et vous pouvez commencer à l’aide de Power BI.

    ![Compte existant](media/service-self-service-signup-for-power-bi/welcome-screen.png)

## <a name="trial-expiration"></a>Expiration de l’essai

Lorsque votre version d’évaluation Power BI Pro arrive à expiration, vos modifications de licence pour une licence (gratuite) Power BI. Une fois cela effectué, vous n’avez plus accès aux fonctionnalités qui nécessitent une licence Power BI Pro. Pour plus d’informations, consultez [Fonctionnalités par type de licence](service-features-license-type.md).

Si une licence (gratuite) Power BI est suffisante, vous n’avez rien à faire. Pour tirer parti des fonctionnalités Power BI Pro, contactez votre administrateur informatique sur l’achat d’une licence Power BI Pro.
