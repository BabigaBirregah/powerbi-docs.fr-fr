# <a name="power-bi-premium-faq"></a>FAQ Power BI Premium
Parcourir une liste de questions (FAQ) et de réponses sur l’offre de Power BI Premium.

* Si vous avez des questions, [essayez d’interroger la communauté Power BI](http://community.powerbi.com/).
* Le problème persiste ? Visitez la [page de support Power BI](https://powerbi.microsoft.com/support/).
* Ou cliquez sur l’icône **Me contacter** ![](media/service-premium-faq/power-bi-contact-me-icon.png) dans le coin inférieur droit de cette fenêtre de navigateur.

## <a name="power-bi-premium"></a>Power BI Premium
**Qu’est-ce que Power BI Premium ?**  
Power BI Premium est une nouvelle offre basée sur la capacité qui inclut :

* la possibilité de diffuser largement des rapports au sein d’une entreprise, sans exiger que les destinataires possèdent une licence individuelle par utilisateur ;
* plus d’évolutivité et de performances à partir d’une capacité dédiée dans le service Power BI ;
* la possibilité de mettre à jour les actifs BI locaux avec Power BI Report Server ;
* une surface d’API, un ensemble cohérent de fonctionnalités et l’accès aux fonctionnalités les plus récentes pour l’analytique intégrée.

**Quelles sont les caractéristiques de Power BI Premium ? Comment fonctionne-t-il ?**  
Power BI Premium consiste en une capacité dans le service Power BI exclusivement allouée à chaque organisation et prise en charge par un matériel dédié entièrement géré par Microsoft. Les organisations peuvent choisir d’appliquer leur capacité dédiée globalement ou l’affecter aux espaces de travail attribués en fonction du nombre d’utilisateurs, des besoins de charge de travail ou d’autres facteurs, et monter ou descendre en puissance suivant l’évolution des besoins.

**En quoi Power BI Pro diffère-t-il de Power BI Premium ? Pourquoi Power BI Premium est-il nécessaire ?**  
La licence Power BI Premium est basée sur la capacité et celle de Power BI Pro sur l’utilisateur.

Power BI Premium consiste en une capacité dans le service Power BI exclusivement allouée à chaque organisation et prise en charge par un matériel dédié entièrement géré par Microsoft : suppression des restrictions sur les tailles de jeu de données et les fréquences d’actualisation, et prise en charge de l’isolation grâce au matériel dédié. Power BI Premium offre également la possibilité de diffuser largement des rapports au sein d’une entreprise, sans exiger que les destinataires possèdent une licence individuelle par utilisateur.

Power BI Pro s’adresse aux utilisateurs qui publient des rapports, partagent des tableaux de bord, collaborent avec des collègues dans des espaces de travail et qui sont impliqués dans d’autres activités connexes, par exemple :

* modification et enregistrement de vues personnalisées ;
* création de tableaux de bord personnels (épingler sur un nouveau tableau de bord) ;
* analyse de données dans Excel ou Power BI Desktop ;
* partage avec prise en charge d’Excel Web App ;
* partage de tableaux de bord et collaboration avec des groupes Office 365 ;
* intégration de contenu avec Microsoft Teams.

**Ai-je besoin de Power BI Pro pour utiliser Power BI Premium ?**  
Oui. Power BI Pro est requis pour publier des rapports, partager des tableaux de bord, collaborer avec des collègues dans des espaces de travail et s’impliquer dans d’autres activités connexes.

**Quel sera le coût de Power BI Premium ? Combien de références (SKU) rendrez-vous disponibles ?**  
Vous pourrez acheter Power BI Premium en fonction du nombre de cœurs virtuels. Pour les tarifs, consultez la [page de tarification de Power BI](https://powerbi.microsoft.com/pricing/). Pour plus d’informations sur les nœuds et les cœurs virtuels, consultez le [livre blanc de Microsoft Power BI Premium](https://aka.ms/pbipremiumwhitepaper). Utilisez également cette [calculatrice de coûts](https://powerbi.microsoft.com/calculator/) pour estimer la capacité Premium dont vous pouvez avoir besoin.

**Qu’entendez-vous par « capacité » ?**  
« Capacité » fait référence à une capacité nommée configurée par un administrateur via le portail de gestion de la capacité Power BI Premium. Une capacité est un regroupement d’un ou plusieurs nœuds. Pour plus d’informations, consultez le [livre blanc de Microsoft Power BI Premium](https://aka.ms/pbipremiumwhitepaper).

**Comment que Power BI Premium est-il facturé ?**  
Power BI Premium est facturé mensuellement sur la base d’un engagement annuel.

**Comment pourrai-je acquérir Power BI Premium ?**  
Power BI Premium est disponible comme module complémentaire pour Power BI Pro dans le centre d’administration Office 365. Pour plus d’informations, consultez la page [Comment acheter Power BI Premium](service-admin-premium-purchase.md). Vous pouvez également contacter votre fournisseur Microsoft pour plus de détails.

**Comment savoir quelle capacité Power BI Premium je dois acheter ?**  
Utilisez cette [calculatrice de coûts](https://powerbi.microsoft.com/calculator/) pour estimer la capacité Power BI Premium dont vous pouvez avoir besoin.

**Quand Power BI Premium sera-t-il disponible ? Il sera disponible simultanément dans toutes les zones géographiques ?**  
Power BI Premium sera généralement disponible à la fin du second trimestre 2017 dans toutes les zones géographiques actuellement prises en charge par le service Power BI.

**BI Premium respectera-t-il la conformité en matière de certification et de réglementation au moment de la mise sur le marché ? Qu’en est-il des clouds nationaux ?**  
Power BI Premium sera généralement disponible à la fin du second trimestre 2017 dans toutes les zones géographiques actuellement prises en charge par le service Power BI. Les clouds nationaux (spécifiquement pour l’Allemagne, la Chine et le gouvernement des États-Unis) seront, selon les prévisions, disponibles au troisième trimestre 2017. Les normes de conformité de Power BI en matière de certification et de réglementation s’appliqueront à Power BI Premium au moment de la disponibilité générale.

**Power BI Premium est-il disponible avec Office 365 E5 ?**  
Power BI Premium est disponible en tant que module complémentaire pour Power BI Pro. Office 365 E5 inclut Power BI Pro. Les clients E5 peuvent acheter Premium en complément de leurs licences Pro existantes.

**Qu’est-ce que Power BI Report Server ? Cela signifie-t-il que Power BI est disponible sur site ?**  
Power BI Report Server est disponible dans le cadre de Power BI Premium et compatible avec les rapports SQL Server Reporting Services, ce qui permet le déploiement et la distribution de rapports Power BI interactifs et de rapports paginés, sur site dans les limites du pare-feu d’une organisation. Les organisations peuvent choisir Power BI dans le cloud ou conserver les rapports localement avec Power BI Report Server et migrer vers le cloud à leur rythme.

Les fonctionnalités de Power BI telles que les tableaux de bord en temps réel, les requêtes en langage naturel et d’autres fonctionnalités du service Power BI ne sont pas disponibles dans le cadre de Power BI Report Server. 

Pour plus d’informations sur Power BI Report Server, voir [Prise en main de Power BI Report Server](report-server/get-started.md).

**Power BI Report Server est-il la fonctionnalité mise à disposition par Microsoft en tant que version d’évaluation technique des rapports Power BI pour SSRS ? Pourquoi fait-il désormais partie de Power BI quand il semble être une fonctionnalité de SQL ?**  
Power BI Report Server est disponible dans le cadre de Power BI Premium et compatible avec les rapports SQL Server Reporting Services, ce qui permet le déploiement et la distribution de rapports Power BI interactifs et de rapports paginés, sur site dans les limites du pare-feu d’une organisation.

Avec l’annonce de Power BI Premium, nous avons pour la première fois divulgué comment cette fonctionnalité sera mise sur le marché.

Pour plus d’informations sur Power BI Report Server, voir [Prise en main de Power BI Report Server](report-server/get-started.md).

**Qu’est-ce que les applications Power BI ?**  
Nous transformons les packs de contenu en applications Power BI pour améliorer la manière dont les utilisateurs découvrent et examinent les informations (insights) à l’échelle de l’entreprise. À présent disponibles, les applications Power BI offrent un moyen simplifié de déployer des tableaux de bord et des rapports pour des personnes spécifiques, des groupes ou l’ensemble d’une organisation. Les utilisateurs professionnels peuvent facilement installer ces applications et les explorer, en centralisant le contenu dans un seul emplacement et en le mettant à jour automatiquement. 

Pour plus d’informations sur les applications, consultez la page [Que sont les applications dans Power BI ?](service-install-use-apps.md).

**Apportez-vous des modifications à Power BI Pro, Power BI gratuit, Power BI Desktop et al. dans le cadre du déploiement de Power BI Premium ?**  
Aujourd’hui, nous simplifions également la distinction entre Power BI Pro et le service gratuit. Selon des commentaires que nous avons reçus, le service gratuit étant conçu pour une utilisation personnelle et Power BI Pro permettant la collaboration, les différences fonctionnelles entre les deux produits ont été une source de confusion pour les utilisateurs. À l’avenir, nous améliorerons le service gratuit pour qu’il offre les mêmes fonctionnalités que Power BI Pro, mais les fonctions de partage et de collaboration seront réservées aux utilisateurs de Power BI Pro.

Les utilisateurs du service Power BI gratuit bénéficieront de l’accès à toutes les sources de données, de l’augmentation des limites de stockage de l’espace de travail, et d’une fréquence d’actualisation et d’une vitesse de diffusion en continu plus élevées. Ces modifications seront effectives à compter du 1er juin 2017 et que vous pouvez en savoir plus en consultant la page [Inscription en libre-service à Power BI](service-self-service-signup-for-power-bi.md). Power BI Desktop reste disponible gratuitement.

Pour plus d’informations concernant l’offre de l’essai Pro prolongé, consultez la page [Activation de l’essai Pro prolongé](service-extended-pro-trial.md).

## <a name="power-bi-free"></a>Power BI (version gratuite)
**Quelles sont les fonctionnalités ajoutées au service gratuit ?**  
À compter du 1er juin 2017, le service gratuit aura des capacités équivalentes à celles de Power BI Pro, à l’exception du partage, de la distribution dans l’entreprise et de la collaboration. Cela inclut la même limite d’espace de travail de 10 Go, jusqu’à 8 actualisations planifiées quotidiennes pour les jeux de données et un débit maximum de données de diffusion en continu égal à 1 M/heure. Nous incluons également l’accès à toutes les sources de données, y compris celles qui sont disponibles via la passerelle de données locale.

**Quelles sont les capacités de Power BI Pro non disponibles avec le service gratuit ?**  
Le partage des tableaux de bord d’égal à égal, les espaces de travail d’application (anciennement appelés espaces de travail de groupe), l’analyse dans Excel et les applications Power BI (à moins d’une prise en charge par la capacité de Premium) sont des fonctionnalités limitées à Power BI Pro.

**Quand ces changements prendront-ils effet ?**  
Les modifications du service gratuit seront effectives à compter du 1er juin 2017.

**Comment les utilisateurs du service gratuit affectés seront-ils prévenus de ces modifications ?**  
Les utilisateurs actifs du service gratuit affectés pars les modifications le 1er juin 2017 recevront une communication par courrier électronique de Microsoft, ainsi que des notifications quand ils se connecteront au service Power BI à compter du 3 mai 2017.

Les utilisateurs existants du service gratuit qui ont été actifs au cours de l’année précédente pourront profiter de l’essai gratuit prolongé de Power BI Pro pendant une période de 12 mois. L’offre fournira toutes les fonctionnalités de Power BI Pro de manière à s’assurer que les utilisateurs disposent d’un délai suffisant pour s’adapter au fonctionnement du service. Pour plus d’informations concernant l’offre d’essai Pro prolongé, consultez la page [Activation de l’essai Pro prolongé](service-extended-pro-trial.md).

**Les utilisateurs perdront-ils l’accès à leur contenu (tableaux de bord, rapports) à ce moment-là ?**  
Non. Ils ne perdront l’accès à aucun contenu téléchargé vers le service Power BI. Le contenu partagé avec d’autres personnes risque de ne plus être accessible aux destinataires au 1er juin 2017. De même, à cette date, ils risquent de ne plus avoir accès au contenu que d’autres personnes auront partagé avec eux.

Les utilisateurs existants du service gratuit ayant été actifs au cours de l’année précédente pourront profiter de l’essai gratuit prolongé de Power BI Pro pendant une période de 12 mois. L’offre fournira toutes les fonctionnalités de Power BI Pro de manière à s’assurer que les utilisateurs disposent d’un délai suffisant pour s’adapter au fonctionnement du service. Pour plus d’informations concernant l’offre d’essai Pro prolongé, consultez la page [Activation de l’essai Pro prolongé](service-extended-pro-trial.md).

**Comment les utilisateurs savent-ils s’ils sont éligibles pour l’essai Pro prolongé de Power BI ?**  
Reportez-vous aux [termes et conditions](https://aka.ms/power-bi-trial) de l’essai Pro prolongé.

Pour plus d’informations concernant l’offre d’essai Pro prolongé, consultez la page [Activation de l’essai Pro prolongé](service-extended-pro-trial.md).

**Que doivent faire les utilisateurs s’ils ne sont pas éligibles pour l’essai Pro prolongé de Power BI ?**  
S’inscrire pour un essai standard de Power BI Pro de 60 jours en utilisant [ce lien](https://app.powerbi.com/signupredirect?pbi_source=web).

**L’introduction de Power BI Premium modifie-t-il les fonctionnalités du service gratuit ?**  
Les utilisateurs du service gratuit ayant accès à la capacité dédiée dans Power BI Premium auront la possibilité de recevoir le contenu qui leur sera distribué par les utilisateurs de Power BI Pro via les applications publiées.

## <a name="power-bi-pro"></a>Power BI Pro
**Qu’est-ce qui change avec Power BI Pro ?**  
Aucune modification n’est apportée à Power BI Pro.

**Que se passe-t-il pour Office ? Power BI Pro est-il toujours inclus dans Office 365 E5 ?**  
Oui. Power BI Pro continuera d’être inclus dans Office 365 E5.

**Ai-je la possibilité de retourner mes licences Power BI Pro si je souhaite passer à Power BI Premium ?**  
Non. Power BI Premium est un module complémentaire pour Power BI Pro ; par conséquent, Premium requiert des licences Pro.

**Pourquoi Power BI Pro est-il nécessaire ? Ne suffit-il as d’acheter Power BI Premium ?**  
La licence Power BI Premium est basée sur la capacité, tandis que celle de Power BI Pro est une licence utilisateur pour des analyses en libre-service rapides et faciles à utiliser, s’adressant aux utilisateurs qui ont des besoins en matière de collaboration, de partage de tableaux de bord, d’analyse ad hoc et de publication de rapports, notamment pour les opérations suivantes :

* modification et enregistrement de vues personnalisées ;
* création de tableaux de bord personnels (épingler sur un nouveau tableau de bord) ;
* analyse de données dans Excel ou Power BI Desktop ;
* partager et affichage de classeurs Excel avec prise en charge de Microsoft Excel Web App ;
* partage de tableaux de bord et collaboration avec des groupes Office 365 ;
* intégration de contenu avec Microsoft Teams.

## <a name="power-bi-embedded"></a>Power BI Embedded
**Quelles sont les nouveautés de Power BI Embedded ?**  
Dans le cadre de Power BI Premium, nous faisons converger Power BI Embedded avec le service Power BI pour offrir une surface d’API, un ensemble cohérent de fonctionnalités et l’accès aux fonctionnalités les plus récentes. Pour l’avenir, nous encourageons les personnes intéressées par l’intégration de Power BI dans leurs applications à commencer par Power BI Desktop et à passer au déploiement avec Power BI Premium.

**Quels sont les avantages de Power BI Premium pour l’intégration de l’analytique dans les applications ?**  
Power BI Premium est une offre basée sur la capacité qui s’exécute sur du matériel dédié et qui est entièrement gérée par Microsoft dans le cadre du service Power BI. Si vous considérez le cycle de développement des applications, Power BI Premium propose de nouveaux avantages, notamment des coûts flexibles et prévisibles, la gestion de contenu améliorée pour les développeurs et la disponibilité complète des fonctionnalités de Power BI pour les scénarios intégrés.

**Ai-je besoin de migrer mon application existante basée sur Power BI Embedded ?**  
Power BI Embedded restera disponible pour les applications existantes selon le programme suivant :

| Contrat de licence | Disponibilité de Power BI incorporé |
| --- | --- |
| Contrat d’entreprise |Jusqu’à l’expiration du contrat existant |
| Direct et fournisseur de solutions cloud (CSP) |Un an à partir de la disponibilité générale de Power BI Premium |

Oui, vous devez commencer à réfléchir à la migration de votre solution analytique intégrée vers Power BI Premium. Pour plus d’informations, consultez la page [Comment migrer le contenu d’une collection d’espaces de travail Power BI Embedded vers Power BI](developer/migrate-from-powerbi-embedded.md).

**Comment procéder pour migrer mon application existante basée sur Power BI Embedded ?**  
Reportez-vous à la [documentation sur la migration](developer/migrate-from-powerbi-embedded.md) pour des conseils sur la migration des applications.

**Microsoft revient-il sur son engagement relatif à Power BI Embedded gratuit ?**  
Avec le modèle précédent, nous proposions un nombre limité de sessions pour le développement et les tests. Dans le nouveau scénario, les clients disposeront toujours d’options pour commencer à développer et effectuer des tests gratuitement à leur guise afin d’être prêts pour la production. Les clients bénéficieront d’un version d’évaluation de Power BI Pro de 2 mois pour la mise en route avec le développement et les tests. Power BI Desktop, l’exploration de données et l’outil de création de rapports resteront disponibles gratuitement.

**Power BI Embedded est-il toujours un service Azure ? Où pourrai-je acheter, configurer et gérer Power BI Embedded ?**  
Power BI Embedded converge avec le service Power BI ; par conséquent, il sera acheté, configuré et géré en tant que tel à l’avenir. Pour plus d’informations, voir [Incorporation avec Power BI](developer/embedding.md).

**Quelles modifications apportez-vous à Power BI Desktop ?**

## <a name="power-bi-report-server"></a>Power BI Report Server
**Pourquoi Power BI Report Server est-il inclus dans Power BI Premium et non vendu séparément ?**  
Power BI Premium introduit la possibilité de gérer les actifs de l’informatique décisionnelle (BI) localement avec Power BI Report Server. Power BI Report Server est un serveur local qui permet le déploiement et la distribution de rapports Power BI interactifs et de rapports paginés complètement dans les limites du pare-feu de l’organisation. Avec Power BI Premium, il est possible de déployer sur site autant de cœurs virtuels qu’une organisation en configure dans le cloud via Power BI Report Server, sans devoir fractionner la capacité. Les organisations peuvent choisir Power BI dans le cloud ou conserver les rapports localement avec Power BI Report Server et migrer vers le cloud à leur rythme.

Pour plus d’informations, voir [Prise en main de Power BI Report Server](report-server/get-started.md).

**Comment les clients peuvent-ils accéder à Power BI Report Server ?**  
Power BI Report Server est disponible via Power BI Premium ou en tant qu’avantage pour les clients ayant acquis SQL Server EE avec SA. Contactez votre fournisseur Microsoft pour plus de détails.

**Pourquoi Power BI Report Server n’est-il pas disponible en tant qu’avantage pour les clients SQL Server Standard ou Enterprise Edition ?**  
Power BI Report Server est disponible pour les clients SQL Server détenteurs de licences EE + SA ou à l’achat via Power BI Premium. Power BI est un service SaaS, et Power BI Report Server suivra le même modèle de livraison basé sur l’abonnement.
